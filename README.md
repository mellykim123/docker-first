# docker-first

##

```shell
git init
yarn set version berry
yarn init # package.json 생성
# package.json 수정
yarn add express
yarn add --dev @types/express typescript nodemon ts-node @types/node
yarn tsc --init # tsconfig.json 생성
# tsconfig.json 수정
yarn add --dev --exact prettier
# .prettierrc.json 수정
# VSCode setting.json 수정
yarn dlx @yarnpkg/sdks vscode
yarn plugin import workspace-tools
# Dockerfile 생성 및 수정
# docker-compose.yml 생성 및 수정
```
